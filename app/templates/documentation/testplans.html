What is?

Test plan - is a list of test cases, that define quality state of main application parts.

Who can create release?

  • Administrator
  • Q&A Engineer

I'm admin/engineer, how i can create it?

In left menu:

  • Click "Create test plan"
  • Fill name and description
  • Click "Commit"

How i can manage testplans?

In left menu:

  • Click "Releases"
  • Click "Current release"
  • Choose Test plan and click by it's name
  • At this point you can:
    • Create testcases
    • Close testplan
    • Control statuses
