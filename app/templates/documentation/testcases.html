What is?

Test case - is a case, that define quality state of application small part .

Who can create test case?

  • Administrator
  • Q&A Engineer

I'm admin/engineer, how i can create it?

In left menu:

  • Click "Releases"
  • Click "Current release"
  • Choose Test plan and click by it's name
  • Click "Create test case"
  • Click "Commit"

I want to edit it, how i can do this?

In left menu:

  • Click "Releases"
  • Click "Current release"
  • Choose Test plan and click by it's name
  • Choose Test case and click by it's name
  • At this point you can
    • Edit test case
    • Delete test case

    • What else i can do?

    Near test case (at right) you can see buttons "Pass"/"Failed".
    So, it's main automaton function - when you define test case state - you click button and result commited to db.
