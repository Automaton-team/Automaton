What is?

Releases - is a essence, that define state of your application state.
How many tests failed/passed, what working well or not and other.

Who can create release?

  • Administrator
  • Release manager (manager)

I'm admin/manager, how i can create it?

In left menu:

  • Click "Releases"
  • Click "Create new"
  • Fill name and description
  • Click "Commit"

How many releases i can create?

Only one at one time.
If you wanna create new release - first close current.

And how i can manage and control current state of release?

In left menu:

  • Click "Releases"
  • Click "Current release"
  • Control state and other things

How i can close release?

In left menu:

  • Click "Releases"
  • Click "Current release"
  • Click "Close release"

Where is information about old releases?

In left menu:

  • Click "Releases"
  • Click "History"
At this page you can see short information about old releases.
Also you can download full report about old release, by click at "download xml" button.
